# Learning Perl Mojolicious & Hypnotoad

## Overview

[Mojolicious](https://www.mojolicious.org/) is a light weight web framework for
Perl.

[Hypnotoad](https://docs.mojolicious.org/Mojo/Server/Hypnotoad) is the web
server that can be used to run production mojo apps.

## Basics

Launch the app using `morbo` to run in dev mode.

```
morbo ./myapp.pl
Web application available at http://127.0.0.1:3000
```
